# Video 4: Refactorización de un Proyecto RAG Legacy con LangChain — Tutorial Guiado

Materiales para el Video 4, demostrando cómo refactorizar un proyecto RAG legacy basado en scripts usando abstracciones de LangChain.

## Contenidos

- `script.md`: Script de narración detallado.
- `slides.md`: Esquema de diapositivas.
- `requirements.txt`: Dependencias Python para demo de refactorización.
- `sample-data/`: Documentos de muestra usados por código legacy y refactorizado.
- `legacy-project/`: Archivos del proyecto RAG legacy original.
- `refactored-project/`: Proyecto RAG refactorizado usando LangChain.
