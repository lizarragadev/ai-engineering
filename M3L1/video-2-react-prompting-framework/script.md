# Video 2: El framework de prompting ReAct — Teoria y practica

**Duracion estimada:** 9 minutos

**Guion de alto nivel:**
1. **Introduccion a ReAct (0:00–1:00):** Definir el framework ReAct, su motivacion y como difiere del prompt engineering estandar.
2. **Estructura del framework (1:00–2:00):** Explicar el ciclo de trabajo ReAct: Pensar → Actuar → Observar → Repetir.
3. **Arquitectura del prompt (2:00–3:00):** Mostrar la anatomia de un prompt ReAct con anotaciones claras.
4. **Recorrido de codigo (3:00–6:30):** Demostrar la implementacion en Python de un agente ReAct que maneja una consulta de varios pasos, enfatizando razonamiento e invocacion de herramientas.
5. **Demo en vivo (6:30–8:00):** Ejecutar el agente con entrada de muestra, narrar los pasos intermedios de razonamiento y accion, y mostrar la salida resultante.
6. **Buenas practicas y errores comunes (8:00–8:45):** Discutir errores comunes como formato incorrecto del prompt y problemas de registro de herramientas, y como evitarlos.
7. **Resumen (8:45–9:00):** Resumir puntos esenciales y transicionar al siguiente tema sobre herramientas personalizadas.
