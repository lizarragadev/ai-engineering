# Produccion de video: Clase 1 - Introduccion a agentes de IA

Este repositorio contiene materiales de produccion para los videos asincronos de la clase en **Modulo 3: Agentes, orquestacion y trazabilidad** del programa de Carrera de Ingenieria de IA. Incluye guiones, diapositivas, codigo de ejemplo, datos de muestra y notas de grabacion para cada video de la **Clase 1: Introduccion a agentes de IA**.

## Videos

- **Video 1:** Razonamiento del agente y uso de herramientas — De pipelines LLM a agentes
- **Video 2:** El framework de prompting ReAct — Teoria y practica
- **Video 3:** Diseno e integracion de herramientas personalizadas en flujos de trabajo de agentes

## Uso

1. Clona este repositorio:
   ```bash
   git clone <repo_url>
   cd M3L1
   ```
2. Navega a la carpeta del video deseado y sigue el `README.md` interno para instrucciones de grabacion, guiones, diapositivas, demos de codigo y datos de muestra.
